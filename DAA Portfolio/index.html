Introduction +

This course covers the core principles of algorithmic problem-solving, focusing on the design and analysis of efficient algorithms. We explore a range of fundamental data structures, including arrays, stacks, queues, trees, and graphs, and learn how to use them effectively within algorithmic solutions. The main role of this the DAA course is to make us understand time and space complexity, enabling us to properly evaluate and select the most suitable algorithm for a given problem based on the algorithm's performance .

Course Overview +

Data Structures

  • Trees: Hierarchical structures with nodes and edges, used for efficient searching and sorting.
    • Binary Trees
    • Binary Search Trees
    • AVL Trees
    • Heaps
  • Graphs: Networks of nodes connected by edges, representing relationships between entities.

Algorithms

Sorting Algorithms:

  • Bubble Sort
  • Insertion Sort
  • Selection Sort
  • Merge Sort
  • Quicksort
  • Heap Sort

Searching Algorithms:

  • Linear Search
  • Binary Search

String Matching Algorithms:

  • Brute Force string matching algorithm
  • Boyre Moore algorithm
  • KMP algorithm
  • Rabin-Karp algorithm

Graph Algorithms:

  • Depth-First Search (DFS)
  • Breadth-First Search (BFS)
  • Dijkstra's Algorithm (shortest path)
  • Floyd Algorithm
  • Warshall Algorithm
  • Bellman Ford Algorithm

Other Algorithms:

  • Dynamic Programming: Breaking down problems into smaller overlapping subproblems.
  • Greedy Algorithms: Making locally optimal choices at each step to find a global optimum.
  • Divide and Conquer: Breaking down problems into smaller subproblems, solving them recursively, and combining the solutions.
